# Dean AL - FAQ Rule-Based System

A rule-based AI FAQ portal for Dedan Kimathi University of Technology (DeKUT), built to help students get instant answers to common academic questions.

## Overview
Dean AL is a Streamlit web application that uses keyword matching and rule-based inference to automatically respond to student queries directed at the Dean's office.

## Features
- Student FAQ browser with search functionality
- Rule-based AI question answering with confidence scoring
- Student question submission portal
- Dean/Admin panel to manage FAQs and view submissions
- Rule management and system statistics dashboard

## How It Works
1. **Keyword Matching** — Student questions are scanned for specific keywords
2. **Rule Matching** — Keywords trigger predefined rules
3. **Confidence Scoring** — System calculates how well the question matches each rule
4. **Best Answer Selection** — The highest confidence match is returned

## Topics Covered
- Appointments with the Dean
- Academic transcripts
- Exam appeals and missed exams
- Study deferment
- Course changes
- Academic complaints
- Special examinations
- Academic advising
- Fees and payments
