# Fuzzy Logic System

A Python-based fuzzy logic inference system that uses membership functions and fuzzy rules to make decisions based on imprecise or uncertain inputs.

## Overview
This system applies fuzzy logic principles to evaluate inputs and derive outputs using a set of linguistic rules, making it suitable for control systems and decision-making applications.

## Features
- Fuzzy membership functions (triangular/trapezoidal)
- Linguistic variable definitions
- Rule-based fuzzy inference
- Defuzzification to produce crisp outputs

## How It Works
1. **Fuzzification** — Crisp inputs are converted into fuzzy values using membership functions
2. **Rule Evaluation** — Fuzzy rules are applied to the fuzzified inputs
3. **Aggregation** — All rule outputs are combined
4. **Defuzzification** — The fuzzy output is converted back to a crisp value
